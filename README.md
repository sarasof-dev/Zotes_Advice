# Zotes-Advice

A simple Python fan project featuring Zote the Mighty and his 57 Precepts from *Hollow Knight*.

## About

This is a small fan project created for Python practice. It features Zote the Mighty in his room in Godhome and includes all of his Precepts.

## Features

- Typewriter text effect
- Animated talking sprite (2 frames)
- Simple Tkinter GUI
- Random precept button
- Precept selector

## Running

```bash
python Zotes_advice.py
```
